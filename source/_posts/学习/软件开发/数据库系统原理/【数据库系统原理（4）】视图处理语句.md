# 【数据库系统原理（4）】视图处理语句

## 创建或修改

```sql
{CREATE|ALTER} VIEW <viewName>[(<columnName>[,...])] as <selectSql>
[WITH CHECK OPTION]
```

## 删除

```sql
DROP VIEW [IF EXISTS] <viewName>
```

## 显示

```sql
SHOW CREATE VIEW <viewName>
```

# 【数据库系统原理（1）】库处理语句

## 连接数据库

```sql
mysql -u <user> -p
<password>
```

## 查看

```sql
SHOW DATABASES;
```

## 创建

```sql
CREATE
DATABASE [IF NOT EXISTS] <name>;
```

## 删除

```sql
DROP
DATABASE <name>;
```

## 选择

```sql
USE <name>;
```

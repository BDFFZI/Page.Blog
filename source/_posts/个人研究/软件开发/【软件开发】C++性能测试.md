## 传递函数

通过变量传递函数并执行

| 速度 | 方式                                          | 分析       |
| ---- | --------------------------------------------- | ---------- |
| 100% | 利用模板传递 lambda 表达式。                  |
| 99%  | 利用函数指针传递已定义函数。                  |
| 98%  | 利用函数指针传递 lambda 表达式。              |
| 50%  | 利用已创建的`std::function`通过引用传递函数。 | 缓存了对象 |
| 20%  | 利用 lambda 隐式转换传递`std::function`引用。 | 未缓存对象 |

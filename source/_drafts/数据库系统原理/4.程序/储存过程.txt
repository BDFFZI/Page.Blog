修改结束符：
delimiter {sign}

参数说明：{in|out|inout} {parameterName} {dataType}

创建储存过程：
create procedure procedureName([参数说明][,])
begin
    [functionBody]
end

局部变量：
声明：declare {variableName}[,] {dataType} [default {value}] 
设置：
set {{variableName}={value}}[,];
select {columnName}[,] into {variableName}[,] ...

条件语句:
if {condition}
then {functionBody}
[else {functionBody}]
end if;

case 
{when {condition} then {functionBody}}[...] 
[else {functionBody}] 
end case;

循环语句：
while {condition} do
{functionBody}
end while;

repeat
{functionBody}
until {condition}
end repeat;

{loopName}:loop
[leave {loopName}]  //break
[iterate {loopName}] //continue
end loop;

游标：
定义：declare {cursorName} cursor for {selectSql};
打开：open {cursorName};
读取：fetch {cursorName} into {variableName}[,];
关闭：close {cursorName};
当游标为空时的回调： declare continue handler for not found set {variableName}={value}; //必须定义在游标后面

调用：
call {procedureName}([parameterValue][,])

删除：
drop procedure [if exists] {procedureName}

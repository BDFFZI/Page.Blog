# 一维线性插值
```
若有
float3 A,B;//需要插值的线段
float3 D;//A,B的插值结果
float t;//插值比率
则
D - A = (B - A) * t
D = Bt - At + A;
D = A(1 - t) + Bt;
```
# 二维线性插值
```
现在增加
float3 C;//和A，B组成三角形的第三个顶点
float3 E;//C,D的插值结果
float3 s//插值比率
根据上文推导可得
E = C(1 - s) + Ds
带入上文公式
E = C(1 - s) + A(1 - t) * s + B * t * s;
设
float α = (1 - s);
float β = (1 - t) * s;
float γ = t * s;
则三角形插值可写作(A,B,C分别任意表示三角形的一个顶点)
E = Aα + Bβ + Cγ;
其中
α + β + γ
= (1 - s) + (1 - t) * s + t * s
= 1 - s + (1 - t + t) * s
= 1 - s + s
= 1

```
